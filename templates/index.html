Breast Cancer Prediction System

AI/ML-powered tool to predict if a breast tumor is Benign or Malignant.

About This Tool

  • Goal: Predict if a breast tumor is cancerous (malignant) or not (benign).
  • Technique: Logistic Regression (supervised classification algorithm).
  • Why: Assist doctors with quick, accurate, and AI-driven predictions using medical data.
  • Inputs: Cell sample measurements (see below).
  • Outputs: Benign (2) or Malignant (4) tumor prediction.

How does it work?
The model uses several features from a breast cell sample, processes them via logistic regression, and predicts whether the tumor is likely benign or malignant. This helps in early detection and treatment planning.
